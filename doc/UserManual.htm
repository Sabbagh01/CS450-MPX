5x5 Operating System User's Manual

User's Manual Version: R3/4



What is in this manual?

This manual goes over all commands that is available to use in 5x5 OS


Commands
R1

Help
Provides information on each command that you need help on.
When selected, it will ask for input from the user on which command they would like information on. The user can give a number input to output the command desired, string input, or input "all" to output all commands.

Help Being Used With A Number


Help Being Used With A String


Help All

Set Time
Sets the local time.
When selected, it will ask for the hour, minute, then second. The hour input must be between 0-23. The hour input must be between 0-23. The second input must be between 0-59. There is no output unless an implausible time is entered, then the system will output "Could not parse, please re-enter time:"

Set Time Command Example

Get Time
Gets the local time.
This command has no inputs. When selected, it will output the local time in Hours:Minutes:Seconds format. Hours being between 00-23, minutes between 00-59, and seconds between 00-59.

Get Time Command Example

Set Date
Sets the local date.
When selected, it will ask for the month, year, then day of the month. The month input must fall between 1-12. The year input must be two digits between 0 and 99. The day input must be an actual date of the chosen month. There is no output unless an implausible date is entered, then the system will output "Could not parse, please re-enter time:"

Set Date Command Example

Get Date
Gets the local date.
This command has no inputs. When selected, it will output the local date as "Month Date, Year" format. The month can be any of the 12 months. The date can be any valid date of the given month. The year can be between 2000-2099.

Get Date Command Example

Version
Shows the current version and compilation date
This command takes no inputs. It will only output the current version as well as the compilation date.

Version Command Example

Shutdown
Shuts down the system
When called, the system will output a confirmation question. If 1 is entered, the system will output "Shutting down now" and then shut down. If anything else is entered, the system will output "Shut down cancelled" and go back to the menu.

Shut Down Command Example

R2

Change PCB Priority
Changes the Priority of a PCB
When called, it will ask the user for the name of the PCB whose priority wants to be changed, then it will ask what priority to set the PCB to. Errors will pop up if the name of the process is invalid or the priority is not within 0-9. The command has no output.

Change PCB Priority Example

Show PCB
Shows a given PCB
When called, the system will ask the user for the name of the PCB to show. If the name is invalid, it will give an error saying there is no PCB. If found, it will output the PCB's name, class, priority, state, and suspended status.

Show PCB Command Example

Show Ready PCB
Shows all ready PCBs
When called, the system will output all ready PCBs that can be found. If none are found, nothing will be outputted.

Show Ready PCB Command Example

Show Blocked PCB
Shows all blocked PCBs
When called, the system will output all blocked PCBs that can be found. If none are found, nothing will be outputted.

Show Blocked PCB Command Example

Show All PCB
Shows all PCBs
When called, the system will output all PCBs that can be found. If none are found, nothing will be outputted.

Show All PCBs Command Example

Delete PCB
Deletes a PCB
When called, the system will ask for the name of the process to delete. If found, it will delete the PCB. Else, it will output an error saying the PCB can not be found. If the given process was found and is a kernel process, it will not delete the PCB and output a message saying it can not.

Delete PCB Command Example


Kernel Delete PCB Command Example

Block PCB
Blocks a PCB
When called, the system will ask for the name of the process to block. If found, it will block the PCB. Else, it will output an error saying the PCB can not be found.

Block PCB Command Example

Unblock PCB
Unblocks a PCB
When called, the system will ask for the name of the process to unblock. If found, it will unblock the PCB. Else, it will output an error saying the PCB can not be found.

Unblock PCB Command Example

Suspend PCB
Suspends a PCB
When called, the system will ask for the name of the process to suspend. If found, it will suspend the PCB. Else, it will output an error saying the PCB can not be found. If the given process was found and is a kernel process, it will not suspend the PCB and output a message saying it can not.

Suspend PCB Command Example


Kernel Suspend PCB Command Example

Resume PCB
Resumes a PCB
When called, the system will ask for the name of the process to resume. If found, it will resume the PCB. Else, it will output an error saying the PCB can not be found.

Resume PCB Command Example


R3

LoadR3
Loads the R3 commands.
When selected, it will insert all the R3 commands into the system. There are no inputs or outputs.

LoadR3 Command Example



R4

Alarm
Sets an Alarm.
When selected, it will ask the user for the month (1-12), last two digits of the year (0-99), day of the month, the hour (0-23), the minute (0-59), the second (0-59), and the message to display. If the user does not provide a valid time, an error will be outputted. At the given time, the alarm will say the given message.

Alarm Command Example

R5

Allocate Memory
Allocates memory
When selected, it will ask the user to enter the size of the memory allocation. If not valid, it will output an error, else it will give the address of the allocated memory.

Allocate Memory Command Example

Free Memory
Frees memory
When selected, it will ask the user for the hexadecimal address of the memory block to be freed. If a non hex input is inserted or the memory block was found it will output an error, else it will say the memory has been freed.

Free Memory Command Example

Show Freed Memory
Shows all freed memory
When selected, it will show the user the address and size of all freed blocks of memory.

Show Free Memory Command Example

Show Allocated Memory
Shows all allocated memory
When selected, it will show the user the address and size of all allocated blocks of memory.

Show Allocated Memory Command Example
