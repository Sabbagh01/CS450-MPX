Commands

Help - Provides information on each command.

Set Time - Sets the time. It will ask for the hour, minute, then second.

Get Time - Displays the current time.

Set Date - Sets the date. It will ask for the month, day, then the last two digits of the year.

Get Date - Gets the date.

Version - Gives the current version as well as the compilation date.

Shut Down - Shuts down the system, enter 1 to confirm shutdown, anything else to cancel.
